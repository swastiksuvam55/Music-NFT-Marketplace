> A Music NFT Marketplace using Blockchain users can buy music as NFT and the commission goes to the artist who posted it

## Install

```sh
npm install
```

## Usage

```sh
npx hardhat --> npm start --> then connect your metamask wallet to the port 8545
```

## Run tests

```sh
npx hardhat test
```
